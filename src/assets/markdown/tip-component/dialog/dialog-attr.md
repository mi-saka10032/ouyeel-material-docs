属性

| 参数    | 说明             | 类型    | 默认值 |
| :------ | :--------------- | :------ | :----- |
| visible | 用于显示隐藏弹框 | Boolean | false  |
| title   | 弹窗标题         | string  | - -    |
| width   | 弹窗宽度         | string  | 50%    |



事件

| 事件名  | 说明                      | 回调参数 |
| ------- | ------------------------- | -------- |
| cancel  | 取消按钮的回调            | - -      |
| confirm | 确定按钮的回调            | - -      |
| close   | 右上角 X 按钮点击关闭回调 | - -      |
