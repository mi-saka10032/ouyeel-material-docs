属性

| 参数        | 说明                         | 类型          | 可选值       | 默认值  |
| :---------- | :--------------------------- | :------------ | :----------- | :------ |
| size        | 尺寸                         | string        | —/small/mini | —       |
| closable    | 是否可关闭                   | boolean       | —            | false   |
| borderColor | 背景边框色                   | boolean       | —            | #cccccc |
| color       | 背景色                       | string        | —            | —       |
| highLight   | 鼠标移入是否高亮显示         | boolean       | —            | true    |
| width       | 最大宽度，溢出部分省略号显示 | number/string | —            | 'auto'  |



事件

| 事件名 | 说明           | 回调参数 |
| ------ | -------------- | -------- |
| close  | 标签关闭时触发 | - -      |
